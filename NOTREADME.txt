Пользователи:
POST /users/login - Осуществляет вход пользователя

Headers
Не требуются

Request
{
	"login": "<Логин от аккаунта>",
	"password": "<Пароль от аккаунта>"
}

Response
{
	"jwt": "<JWT Ключ>"
}

Errors
400 - Логин или пароль не указаны
401 - Не верный логин или пароль

GET /users/me - Получение пользователя по токену

Additional
Требуется авторизация

Request
Не требуется

Response
{
	"jwt": "<JWT Ключ>",
	"user": {
		# UserModel #
	}
}

Errors
Нету своих

Документы:

-------
Авторизация происходит по JWT Токену переданому в Header`е Authorization пример ниже
Authorization: Bearer <JWT Ключ>

Ошибки авторизации
400 - Токен не указан
401 - Токен не верный
